Warp Grid

Aug 10, 2021

Warp Grid - A component for warping, cropping and re-colouring sprites directly in the Unity editor.

Warp Grid is a simple component for displaying a texture with some warping applied to it via a grid/lattice system, which can be edited directly in the Scene editor window with an intuitive set of brushes. Warp Grid can be used in 3D scenes, but the warping only happens on the local 2D plane of the object the component is added to.

It works by allowing you to subdivide a grid into as many rows and columns as you need. The points in the grid can then be moved around, sculpted, and coloured with various types of brushes.
